26.删除有序数组中的重复项

分析

题目要求：
我们 原地 删除重复出现的元素，使每个元素只出现一次， 并且返回删除后数组的新长度。
不要使用额外的数组空间，你必须在原地修改输入数组并在使用 [O(1)] 额外空间的条件下完成。

[O(1)] 当一个算法的空间复杂度为一个常量，即不随被处理数据量n的大小而改变时，可表示为O(1)。

想法

既然要求使用O(1)额外空间，那就不能再创建新数组，所以我们只能使用双指针。

我一开始是想正判断,但是发现这样做只能满足小部分条件，贴一下我的最初代码

class Solution {
public int removeDuplicates(int[] nums) {
int i=0;

    //while循环也可以
    for(int j = 1;j<nums.length;j++){
        if(nums[j] != nums[i]){
            i++;
            nums[i] = nums[j];
        }
    }
    return i + 1;
}

}

```
