hexo部分异常报错

发表于 2021-03-30 分类于 日常

hexo的异常报错

总结一下在使用hexo搭建博客时候的奇怪报错问题

  • 语言变成了鸟语阿拉伯文

说实话百度有时候是真的不可靠，我在百度上查到的设置hexo主题文字为汉语的方法

均是language: zh-Hans

然后就出现了上面的页面。。。。。。
az

然鹅当我找到Next主题下的语言目录Blog\node_modules\hexo-theme-next\languages

根本就没有zh-Hans啊！！

好了那就把zh-Hans 改成zh-CN吧，果然可以了
博客终于变成自己能看懂的样子了

  • 看看第二个问题

???az

说实话这个问题我一直没有搞懂，但是每次出现这个异常都是在我挂着小飞机 科学上网 的时候才会出现
我一度认为这是小飞机的问题（并不是

直到今天早上我更想的时候，我没有挂着小飞机还是出现了
一顿百度之后，原来是Node.js版本过高

？原来14.16的版本也算高吗，那我之前 科学上网 就不行的时候是怎么回事，难道说Node.js也要看小飞机的脸色行事？ 

(node:1081) Warning: Accessing non-existent property 'lineno' of module exports inside circular dependency
(Use `node --trace-warnings ...` to show where the warning was created)
(node:1081) Warning: Accessing non-existent property 'column' of module exports inside circular dependency
(node:1081) Warning: Accessing non-existent property 'filename' of module exports inside circular dependency
(node:1081) Warning: Accessing non-existent property 'lineno' of module exports inside circular dependency

此外我发现，在command hexo g的时候，会报以上错误，百度了一下果然是Node.js的问题

但我还是没有想懂和小飞机有什么关系
啊，未解之谜

所以解决方法就是降级Node.js的版本，我在这里找到了12.21版本
Node.js v12.21

自己下载重新安装即可

总结

好了目前遇到的问题就只有这两个，要是还有其他问题我会再补充的 希望不要再有问题了
